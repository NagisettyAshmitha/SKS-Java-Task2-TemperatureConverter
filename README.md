# Temperature Converter

## Project Overview

A Temperature Converter Application developed using Java and Web Technologies.

This application converts:

* Celsius to Fahrenheit
* Fahrenheit to Celsius

---

## Technologies Used

* Java
* HTML
* CSS
* JavaScript
